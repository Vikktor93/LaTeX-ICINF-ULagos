\begin{abstract}
Un solo párrafo de máximo 350 palabras que presenta de forma clara y motivadora el proyecto.  No debe incluir citas, referencias o conclusiones detalladas. 
Debe ser preciso:
\begin{itemize}\justifying
  \item El \textbf{problema}: ¿Qué necesidad o problema se aborda? 
\item \textbf{Importancia}: ¿Por qué es interesante o relevante resolverlo? 
\item \textbf{Logros} y \textbf{desafíos}: ¿Qué se consiguió y cuáles fueron los principales retos? 
\end{itemize}

\begin{ejemplo}
El actual proceso de inscripción de asignaturas en la Universidad X es manual y propenso a errores, generando demoras y sobrecarga administrativa. Este proyecto aborda dicha problemática mediante el desarrollo de un sistema web centralizado que automatiza el registro académico. El principal logro es una plataforma funcional que reduce los tiempos de inscripción en un 80\% y garantiza la integridad de los datos. El mayor desafío fue la integración con los sistemas legados de la universidad, superado mediante la creación de una API RESTful.
\end{ejemplo}


\keywords{incluir 5, palabras clave, que describan, el proyecto}
\end{abstract}
